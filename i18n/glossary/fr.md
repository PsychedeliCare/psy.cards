## UI conventions

- Warnings: use imperative ("Utilisez un kit de test", not "Il faudrait utiliser…")
- Keep "GHB/GBL", "LSD", "MDMA", "DMT" as-is
- Translate "test kit" → "kit de test"

## Terms to avoid

- "drogue dure/douce" (moralizing) — prefer "substance" or the specific class name
- Over-clinical phrasing when plain language suffices

## False friends

- "unsafe" in TripSit context → "Risqué" (not "non sécurisé" in isolation)
- "trip" → "expérience" or "effet" depending on context; "trip report" → "récits d'expérience"
